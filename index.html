TwitterBot or Not

"TwitterBot or Not" was made to create a dataset for the Twitter Analysis Tool Senior Design project at Drexel University.
TAT uses machine learning to detect malicious bots on Twitter, we need to create a dataset to see if our tool can beat a human.
